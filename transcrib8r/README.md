### Overview
Transcrib8r is a collaborative AI project focused on transcribing audio content and generating structured study materials for students.

### My Contributions
- Developed backend routes using Python and Flask
- Integrated transcription and generative AI services
- Collaborated in a shared Git codebase for feature iteration and debugging

### Tech Stack
Backend: 
- Python + Flask (CORS)
- Config: .env via python-dotenv (GROQ_API_KEY, OPENAI_API_KEY)

AI Integration:
- Groq Whisper (whisper-large-v3) for transcription
- OpenAI Chat API for study notes

Frontend:
- HTML + JS + CSS

### Note on Code Availability
The source code for this project is maintained in a private team repository and is not publicly available due to collaboration agreements and sensitive information.
